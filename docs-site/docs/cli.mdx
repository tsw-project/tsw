# CLI

Run commands from the root of your MapleStory Worlds project.

## `tsw init`

Creates a `tsconfig.json` and type declarations in a MapleStory Worlds directory.

```bash
tsw init
```

## `tsw build`

Builds TypeScript scripts into MapleStory Worlds `.mlua` output.

```bash
tsw build
```

## `tsw watch`

Watches source files and rebuilds while you work.

```bash
tsw watch
```
